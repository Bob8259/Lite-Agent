# Roo Code (Modified)

This project is modified based on [Roo Code](https://github.com/RooCodeInc/Roo-Code.git).

The original Roo Code consumes too many tokens due to verbose prompts, and it was inconvenient to switch between different modes. This fork addresses those issues.

## Main Changes

- Simplified the prompts to save tokens while ensuring performance
- Added keyboard shortcut `Alt+Q` for quick mode switching
